An open-source kernel written from scratch in C and Assembly, designed for performance, modularity, and as the foundation for the Noble operating system family.

## About

Rather than being tied to a single operating system, NobleKernel is designed to power multiple specialized systems while sharing a common core.

---

## Philosophy

NobleKernel is built around a few simple principles:

- Performance First
- Modular Design
- Open Source
- Educational
- Modern Hardware Support
- Low Latency
- Clean and Maintainable Code

## Goals

- Build a reusable kernel.
- Support multiple operating systems.
- Keep the code modular and clean.
- Learn low-level systems programming.
- Document everything along the way.
